Платформа Docker

  • docker pull [имя образа] - копирование образа приложения.
  • docker images - вывод списка образов.
  • docker run -dp [порт для вызова контейнера]:8080 [имя образа] - запуск контейнера с образом.
  • docker ps - выводит список всех запущеных контейнеров.
  • docker stop id_[id контейнера] - остановка указанного контейнера.
  • docker restart id_[id контейнера] - перезапуск контейнера.
  • docker image rm -f id_][id образа] - удаление образа.
